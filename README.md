# Clean CMS Implementation

This repository contains a quick implementation of a clean content management system (CMS).

## Features
- User authentication
- Basic content creation and editing
- Responsive design

## Tech Stack
- HTML
- CSS
- JavaScript

## Usage
1. Clone the repository.
2. Open the `index.html` file in your browser to see the CMS in action.
